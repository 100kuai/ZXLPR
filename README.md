# ZXLPR
iOS车牌识别，识别率高达98%，仅提供API调动，后台车牌识别系统暂不开源。

## 车牌识别综合测评
经测试，在车牌定位、大角度下、夜间环境下、极端环境下、支持最小车牌像素宽高等测试下，都表现出来识别快、识别率高、使用更便捷的特点。
